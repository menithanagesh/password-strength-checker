# Password Strength Checker (Python)

This project checks the strength of a password based on security rules.

## Description
The program evaluates password strength based on:
- Password length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

The password is classified as Weak, Medium, or Strong.

## Technologies Used
- Python

## How to Run
python password_checker.py

This project was developed as part of a cybersecurity internship task.
